# Personal Assistant reliability regressions

This matrix separates runtime watchdog signals from CI-only UI invariants. Existing owner tests remain the single source of automated coverage; extend them instead of copying scenarios into a second suite.

| Task | User-visible symptom | Invariant | Exact automated test | Watchdog signal | Status |
| --- | --- | --- | --- | --- | --- |
| REG-001 | An attachment fails with `session not found` after Desktop restarts. | Sending an attachment after relaunch resumes the selected durable chat through its owner profile, survives one additional backend replacement, uploads exactly once, and submits to that same chat. | `apps/desktop/src/app/session/hooks/use-prompt-actions/index.test.tsx` — `rebinds once more when a backend relaunch invalidates the first recovered runtime before retry` | `session_not_found` | Covered |
| REG-002 | The current cwd-less conversation disappears from Projects. | Projects keeps the selected cwd-less chat visible without surfacing unrelated loose sessions. | `apps/desktop/src/app/chat/sidebar/projects/workspace-groups.test.ts` — `keeps the selected cwd-less session visible beside Projects without surfacing unrelated loose sessions` | Test gate only; no safe runtime UI observer yet. | Covered |
| REG-003 | Opening Personal Assistant changes the visible sidebar profile. | Activating the owner profile for Personal Assistant never replaces the selected sidebar scope. | `apps/desktop/src/app/session/hooks/use-session-actions.test.tsx` — `opens the global personal assistant without replacing the visible profile scope` | Test gate only; no safe runtime UI observer yet. | Covered |
| REG-004 | Situation opens expanded on every visit. | Situation starts collapsed and expands only after explicit user action. | `apps/desktop/src/app/chat/personal-assistant-situation.test.tsx` — `starts collapsed and expands on demand` | Test gate only; no safe runtime UI observer yet. | Covered |
| REG-005 | Raw partial `hermes-ui` JSON appears while a response streams. | Incomplete JSON shows the preparation state while streaming and never exposes raw partial JSON. | `apps/desktop/src/components/assistant-ui/embeds/hermes-ui-artifact.test.tsx` — `hides incomplete hermes-ui JSON while the form is still streaming` | Test gate only; no safe runtime UI observer yet. | Covered |
| REG-006 | A current lazy backend is reported as out of date. | New and already-live lazy sessions include the current Desktop contract marker in `session.info`. | `tests/test_tui_gateway_server.py` — `test_session_create_lazy_info_reports_desktop_contract`; `test_live_lazy_session_info_reports_desktop_contract` | Test gate; missing-runtime RPCs are monitored separately. | Covered |
| REG-007 | An unread badge persists after the update is actually read, or clears before it is read. | Only a visible, bottomed transcript acknowledges reading; backend reset is idempotent and preserves concurrent updates. | `apps/desktop/src/app/chat/personal-assistant-situation.test.tsx` — `acknowledges unread activity when the open assistant is visible at the bottom`; `tests/tui_gateway/test_personal_assistant_home.py` — `test_home_preserves_unread_activity_until_the_transcript_is_read`, `test_read_acknowledgement_is_idempotent`, `test_read_acknowledgement_retries_a_concurrent_state_change` | Test gate only; no safe runtime UI observer yet. | Covered |
| REG-008 | Opening an old chat displays a different conversation. | A cross-wired or recycled runtime mapping is rejected and the requested durable session is resumed in full. | `apps/desktop/src/app/session/hooks/use-session-actions.test.tsx` — `rejects a cross-wired runtime mapping and falls through to a full resume` | Test gate; `session_not_found` catches failed runtime recovery. | Covered |

Runtime incidents retain the signal, safe session identity, and process snapshot without prompt content. CI failures should report the regression task ID, exact test, command, and captured output. A task is green only when its exact owner test passes; generic suite success is not a substitute.
